Vue实例生命周期

生命周期钩子函数

每个Vue实例在被创建时都要经过一系列的初始化过程

  • 生命周期分为三大阶段：初始化显示、显示更新、销毁Vue实例

    • 初始化阶段的钩子函数：

      beforeCreate()实例创建前：数据和模板均未获取到

      created()实例创建后：最早可以访问到data数据，但模板未获取到

      beforeMount()数据挂载前：模板已获取到，但是数据未挂载到模板上

      mount()数据挂载后：数据已挂载到模板中

    • 更新阶段的钩子函数：

      beforeUpdate()模板更新前：data改变后，更新数据模板前调用

      updated()模板更新后：将data渲染到数据模板中

    • 销毁阶段的钩子函数：

      beforeDestroy()实例销毁前

      destroyed()实例销毁后

liveServer服务端插件

  1. 在 VS Code中安装liveServer服务端插件，用于Ajax接口调用

  2. 启动服务器：

    方式1：任意打开一个html页面，在最下面可看到 Go Live 点击它可启动，默认端口5500

    方式2：在html页面单击右键，点击Open with Live Server 访问页面

    注意：如果之前启动过服务器，则使用 方式2 启动html页面，不能使用 方式1 会端口占用

  3. 更改liveServer默认端口号：

    在设置中进入 setting.json，在json文件中添加"liveServer.settings.port": 8080,
