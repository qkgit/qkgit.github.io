项目介绍与演示

  • ToDoMVC是一个非常经典的案例，功能非常丰富，并且针对多种不同技术分别都开发了此项目，比如：React、AugularJS、JQuery等等。

下载与导入模板

ToDoMVC 模板 下载方式

  1. 使用git克隆项目

  2. 直接下载zip压缩包

    在GitHub下载ToDoMVC 模板在：https://github.com/tastejs/todomvc-app-template

npm 安装依赖

因为下载的模板没有样式，所以要通过npm安装相关依赖，依赖配置在package.json文件中

  • 通过 cmd 进入ToDoMVC项目所在目录，执行命令：npm install

    注意：要使用 npm 命令，需要安装 node.js 环境才能用。

  • 依赖安装完成后，生成的node_modules目录效果如下：

导入到VS Code

  1. 将 todomvc-app-template 目录重命名为： vue-03-todomvc

  2. 拷贝 vue-03-todomvc 拷贝到 D:\StudentProject\WebStudy 目录下

  3. 在 VS Code 左侧资源管理窗口刷新一下就可以看到 vue-03-todomvc ，直接访问 index.html，效果如下：

功能分析

  • 有数据

    • 列表中的记录有三种状态且<li>样式不一样“

      未完成（没有样式)、已完成（.completed）、编辑中（.edliting）

    • 任务字段：id（主键）、content（内容）、completed（状态；true 已完成，false 未完成）

  • 无数据

    • .main和.footer标识的标签应该被隐藏（v-show）

功能分析

  1. 在最上面的文本框中添加新的任务

  2. 不允许添加非空数据。
